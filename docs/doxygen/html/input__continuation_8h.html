Detailed Description

Shared multiline input continuation system.

Provides shared multiline parsing functionality for both the Line Editing Engine (LLE) and the main input system. Handles shell construct detection, quote tracking, bracket counting, and continuation line analysis.

This is shared infrastructure used by multiple components of Lusush.
