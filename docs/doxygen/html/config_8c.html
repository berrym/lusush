Detailed Description

Configuration System Implementation for Lusush Shell.

Handles loading, parsing, and managing shell configuration files. Supports TOML-style configuration with sections, type validation, and runtime updates.

Author
Michael Berry trism.nosp@m.egus.nosp@m.tis@g.nosp@m.mail.nosp@m..com
