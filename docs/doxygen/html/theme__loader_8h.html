Detailed Description

LLE Theme File Loader - File I/O and Theme Loading API.

Specification: Issue #21 - Theme File Loading System Version: 1.0.0

Provides file loading, directory scanning, hot reload, and theme export functionality for the LLE theme system.

Theme File Locations (checked in order):

  1. $XDG_CONFIG_HOME/lusush/themes/ (~/.config fallback)
  2. /etc/lusush/themes/ (system-wide, optional)
