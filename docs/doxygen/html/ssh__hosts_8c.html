Detailed Description

SSH Host Cache and Parsing for LLE Completion.

Provides SSH host completion by parsing:

  • ~/.ssh/config for Host entries
  • ~/.ssh/known_hosts for previously connected hosts

This module is used by the LLE completion system to provide SSH hostname completions for ssh, scp, rsync commands.
