Detailed Description

LLE Theme File Parser - TOML-subset Parser for Theme Files.

Specification: Issue #21 - Theme File Loading System Version: 1.0.0

Custom TOML-subset parser for parsing theme configuration files. Designed to be dependency-free and focused on the subset of TOML needed for theme files.

Supported TOML Features:

  • Key-value pairs: key = "value"
  • Sections: [section] and [section.subsection]
  • Inline tables: { key = "value", other = 123 }
  • Arrays: ["item1", "item2"]
  • Booleans: true, false
  • Integers: 123, -45
  • Strings: "quoted" with
    , \, " escapes
  • Comments: # comment
