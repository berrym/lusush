3 * @brief LLE Theme File Loader - File I/O and Theme Loading API
4 *
5 * Specification: Issue #21 - Theme File Loading System
6 * Version: 1.0.0
7 *
8 * Provides file loading, directory scanning, hot reload, and theme export
9 * functionality for the LLE theme system.
10 *

83/**

110/**

114 * Successfully loaded themes are registered with the registry.

117 * @param registry Theme registry to register themes with

125/**

128 * Loads themes from:
