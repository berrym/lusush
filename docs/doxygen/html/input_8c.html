Detailed Description

Lusush Input System using GNU Readline.

This module provides unified input handling for both interactive and non-interactive modes, with complete GNU readline integration for interactive sessions.

UTF-8 Support: This module uses LLE's UTF-8 support to properly handle multi-byte characters. While shell syntax characters (quotes, brackets, etc.) are all ASCII, we must properly skip over UTF-8 multi-byte sequences to avoid misinterpreting continuation bytes as syntax characters.
