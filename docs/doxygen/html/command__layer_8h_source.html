1/**
2 * @file command_layer.h
3 * @brief Command layer - Real-time syntax highlighting system
4 *
5 * Provides real-time syntax highlighting for command input that works
6 * universally with any prompt structure.
7 *
8 * @author Michael Berry <trismegustis@gmail.com>
9 * @copyright Copyright (C) 2021-2026 Michael Berry
10 * @license GPL-3.0-or-later
11 *
12 * Key Features:
13 * - Real-time syntax highlighting for shell commands
14 * - Universal compatibility with any prompt structure
15 * - Integration with existing lusush syntax highlighting functions
16 * - High-performance command input processing (<5ms updates)
17 * - Memory-safe command text management
18 * - Intelligent cursor position tracking
19 * - Event-driven communication with prompt layer
20 *
21 * Design Principles:
22 * - Independent from prompt structure and layout
23 * - No interference with prompt display or positioning
24 * - Leverages existing proven syntax highlighting logic
25 * - Performance-optimized for real-time updates
26 * - Memory-safe with proper resource management
27 * - Event-driven coordination with other layers
28 *
29 * Strategic Innovation:
30 * This layer completes the revolutionary combination of professional themes
31 * with real-time syntax highlighting by providing command input processing
32 * that works universally with the prompt layer's universal compatibility.
33 */
