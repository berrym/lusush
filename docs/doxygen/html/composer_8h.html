Detailed Description

LLE Prompt Composer - Template/Segment/Theme Integration.

Specification: Spec 25 - Prompt Composer Version: 1.1.0

The prompt composer integrates the template engine, segment registry, and theme registry to render complete prompts. It provides the callbacks needed by the template engine to resolve segments, check visibility, and apply theme colors.

Shell Event Integration (Spec 26): The composer registers handlers with the shell event hub to automatically respond to directory changes, pre-command, and post-command events. This enables event-driven cache invalidation instead of time-based polling.
