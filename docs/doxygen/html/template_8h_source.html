2 * @file template.h
3 * @brief LLE Prompt Template Engine - Type Definitions and Function
4 * Declarations
5 *
6 * Specification: Spec 25 Section 6 - Template Engine
7 * Version: 1.0.0
8 *
9 * The template engine provides expressive format strings for prompt generation
10 * with segment references, conditionals, color application, and property
11 * access.
12 *
13 * Template Syntax:
14 * ${segment} - Render segment (e.g., ${directory}, ${git})
15 * ${segment.property} - Access segment property (e.g., ${git.branch})
16 * ${?segment:true:false} - Conditional: if segment visible, show true
17 * ${?segment.prop:t:f} - Conditional on property existence
18 * ${color:text} - Apply theme color to text
19 * \n - Literal newline
20 * \\ - Escaped backslash
21 * \$ - Escaped dollar sign
22 */
