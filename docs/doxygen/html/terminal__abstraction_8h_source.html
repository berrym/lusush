1/**
2 * @file terminal_abstraction.h
3 * @brief LLE Terminal State Abstraction Layer - Type Definitions
4 *
5 * Specification: Spec 02 - Terminal Abstraction Complete Specification
6 * Version: 1.0.0
7 *
8 * This header contains ALL type definitions and function declarations for the
9 * LLE terminal abstraction system. NO implementations are included here.
10 *
11 * Critical Design Principles:
12 * 1. NEVER query terminal state during operation - internal model is
13 * authoritative
14 * 2. NEVER send direct escape sequences - all output through Lusush display
15 * 3. NEVER assume terminal cursor position - calculate from buffer state
16 * 4. NEVER track terminal state changes - generate complete display content
17 * 5. Internal buffer state is authoritative - single source of truth
18 *
19 * Architecture: Research-validated design following proven patterns from
20 * JLine, ZSH ZLE, Fish, and Rustyline.
21 */
