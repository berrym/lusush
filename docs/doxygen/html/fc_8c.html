Detailed Description

POSIX fc Command Builtin Implementation using LLE History System.

This module implements the POSIX fc (fix command) builtin for Lusush shell. The fc command provides POSIX-compliant history editing, listing, and re-execution capabilities using LLE's history system.

POSIX fc command syntax:

  • fc [-r] [-e editor] [first [last]] # Edit and re-execute
  • fc -l [-nr] [first [last]] # List commands
  • fc -s [old=new] [first] # Substitute and re-execute
