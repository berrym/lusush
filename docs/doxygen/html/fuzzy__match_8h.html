Detailed Description

Unicode-aware fuzzy string matching library.

Provides fuzzy string matching algorithms with full Unicode support. Used by both the shell's autocorrect system and LLE's completion/history search.

Features:

  • Levenshtein (edit) distance
  • Damerau-Levenshtein distance (with transpositions)
  • Jaro-Winkler similarity
  • Subsequence matching
  • Combined weighted scoring
  • Unicode NFC normalization support
  • Case-insensitive matching with Unicode case folding
