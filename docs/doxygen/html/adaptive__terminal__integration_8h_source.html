1/**
2 * adaptive_terminal_integration.h - Adaptive Terminal Integration System
3 *
4 * Provides universal terminal compatibility through detection and control
5 * separation. Enables LLE to work optimally in any environment from
6 * traditional TTY terminals to modern editor interfaces and AI assistants.
7 *
8 * Key Innovation: Detection and Control Separation
9 * - Interactive capability detection operates independently
10 * - Control method selection based on detected capabilities
11 * - Universal compatibility with graceful degradation
12 *
13 * Specification:
14 * docs/lle_specification/critical_gaps/26_adaptive_terminal_integration_complete.md
15 * Date: 2025-11-02
16 */
