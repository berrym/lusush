I/O Redirection Implementation for Lusush Shell

Comprehensive implementation of POSIX shell I/O redirection including:

  • Basic output redirection (>)
  • Append redirection (>>)
  • Input redirection (<)
  • Error redirection (2>, 2>>)
  • Combined redirection (&>)

  • Here documents (<<, <<-)
