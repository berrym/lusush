Detailed Description

Shell builtin command implementations.

Comprehensive implementation of shell builtin commands including cd, echo, export, pwd, exit, jobs, fg, bg, history, config, debug, and many more. Provides POSIX-compliant builtins alongside Lusush-specific extensions.
