Detailed Description

LLE Theme Registry System - Type Definitions and Function Declarations.

Specification: Spec 25 Section 4 - Theme Registry System Version: 1.0.0

The theme registry provides unified storage for all themes (built-in and user). Both types use identical registration, ensuring first-class citizenship for all.

Key Features:

  • Theme registration and lookup
  • Theme inheritance with cycle detection
  • Color schemes with semantic colors
  • Symbol sets with Unicode/ASCII fallbacks
  • Prompt layout templates
