5 * raw terminal data into structured events for the LLE event system.

13 * - Sub-millisecond parsing (<250μs target)
14 * - Intelligent sequence detection and disambiguation
15 * - Comprehensive error recovery
16 * - Event system integration
17 * - Keybinding integration (<10μs lookup)
18 * - Widget hook triggers
