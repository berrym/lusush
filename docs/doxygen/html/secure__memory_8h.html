Detailed Description

Portable Secure Memory Operations.

Provides portable implementations of:

  • Secure memory wiping (prevents compiler optimization)
  • Memory locking (prevents swapping to disk)

Implementation: Spec 15 Minimal Secure Mode Based on: MLOCK_PORTABILITY_ANALYSIS.md Date: 2025-11-06

◆ lle_memory_lock()

bool lle_memory_lock ( void * addr,
size_t len )
inlinestatic

Lock memory to prevent swapping to disk

Uses mlock() on POSIX systems. On systems without mlock or when mlock fails (insufficient privileges), returns false but this is not a fatal error - the buffer can still be used, just without the anti-swap protection.

Parameters
addrAddress of memory to lock
lenLength in bytes
Returns
true if locked successfully, false otherwise
