Detailed Description

LLE User Keybinding Configuration System Implementation.

Implements user-customizable keybindings via TOML configuration files. Reuses the TOML-subset parser from the theme system.

Specification: Spec 13 (User Customization), Spec 22 (User Interface) Version: 1.0.0
