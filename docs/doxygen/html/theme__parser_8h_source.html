1/**
2 * @file theme_parser.h
3 * @brief LLE Theme File Parser - TOML-subset Parser for Theme Files
4 *
5 * Specification: Issue #21 - Theme File Loading System
6 * Version: 1.0.0
7 *
8 * Custom TOML-subset parser for parsing theme configuration files.
9 * Designed to be dependency-free and focused on the subset of TOML
10 * needed for theme files.
11 *
12 * Supported TOML Features:
13 * - Key-value pairs: key = "value"
14 * - Sections: [section] and [section.subsection]
15 * - Inline tables: { key = "value", other = 123 }
16 * - Arrays: ["item1", "item2"]
17 * - Booleans: true, false
18 * - Integers: 123, -45
19 * - Strings: "quoted" with \n, \\, \" escapes
20 * - Comments: # comment
21 */
