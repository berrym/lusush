1/**
2 * @file prompt_layer.h
3 * @brief Prompt layer - Universal prompt rendering system
4 *
5 * Provides universal prompt rendering that works with any prompt structure
6 * without requiring parsing or modification. Integrates with the theme system.
7 *
8 * @author Michael Berry <trismegustis@gmail.com>
9 * @copyright Copyright (C) 2021-2026 Michael Berry
10 * @license GPL-3.0-or-later
11 *
12 * Key Features:
13 * - Universal prompt structure compatibility
14 * - Integration with existing theme system (6 professional themes)
15 * - Event-driven communication with foundation layers
16 * - High-performance rendering with intelligent caching
17 * - Memory-safe prompt content management
18 * - Real-time theme switching support
19 *
20 * Design Principles:
21 * - No prompt structure parsing required
22 * - Works with any prompt format (simple, complex, ASCII art)
23 * - Preserves existing prompt generation functionality
24 * - Performance-optimized (<5ms rendering time)
25 * - Memory-safe with proper resource management
26 *
27 * Strategic Innovation:
28 * This layer enables the revolutionary combination of professional themes
29 * with real-time syntax highlighting by providing universal prompt
30 * compatibility without interfering with command input processing.
31 */
