Detailed Description

Completion Menu Navigation Logic.

Provides navigation and interaction logic for the completion menu. All functions work with menu state; no rendering is performed here. Rendering is handled by the display layer per LLE architecture.

Navigation Actions:

  • Single item: up, down
  • Paging: page up, page down
  • Category jumping: next category, previous category
  • Selection: first, last

All navigation respects the current menu configuration (scrolling, visible items, etc.) and updates the menu state accordingly.
