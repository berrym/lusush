LLE Theme Registry System Implementation.

Specification: Spec 25 Section 4 - Theme Registry System Version: 1.0.0

Implements theme registry, theme creation, inheritance resolution, and built-in themes.

lle_theme_t * lle_theme_create_powerline ( void )

Create the powerline theme.

Powerline-inspired prompt with separator glyphs. Requires a powerline-patched font (Nerd Fonts, Powerline fonts, etc.)

Note: True powerline with colored background segments would require additional composer support for background colors. This theme provides a powerline-style look with foreground colors and separator symbols.

Powerline characters used: U+E0B0 - Left-pointing solid arrow U+E0B1 - Left-pointing thin arrow
