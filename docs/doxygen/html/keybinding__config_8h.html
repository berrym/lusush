Detailed Description

LLE User Keybinding Configuration System.

Provides user-customizable keybindings via configuration files. Users can override default Emacs keybindings by creating: ~/.config/lusush/keybindings.toml

Example config: [bindings] "C-a" = "end-of-line" # Swap C-a and C-e "C-e" = "beginning-of-line" "M-p" = "history-search-backward" "C-s" = "none" # Unbind

Specification: Spec 13 (User Customization), Spec 22 (User Interface) Version: 1.0.0
