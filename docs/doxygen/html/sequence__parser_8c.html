lle_result_t lle_sequence_parser_check_timeout ( lle_sequence_parser_t * parser,
uint64_t timeout_us,
lle_parsed_input_t ** parsed_input )

Check if parser has timed out waiting for sequence completion. If in ESCAPE state and timeout exceeded, returns the ESC as a standalone key.

Parameters
parserThe sequence parser
timeout_usTimeout in microseconds (typically 50000-100000 for ESC)
parsed_inputOutput: If timeout occurred, contains ESC key event
Returns
LLE_SUCCESS if timeout handled, LLE_ERROR_NOT_FOUND if no timeout
