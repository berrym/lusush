adaptive_context_initialization.c - Adaptive Context Initialization

Provides unified initialization and lifecycle management for adaptive terminal integration contexts. Handles detection, controller selection, and interface creation.

Key Features:

  • Automatic mode detection and controller selection
  • Unified interface creation
  • Complete lifecycle management
  • Configuration recommendations
  • Health monitoring and fallback
